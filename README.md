Визуальный редактор графов с возможностью создания узлов, связей, поиска кратчайших путей и визуализации результатов.

## 📋 Возможности

- **Создание узлов** - добавляйте узлы на сцену одним кликом
- **Создание связей** - соединяйте узлы направленными связями с указанием веса
- **Редактирование** - перемещайте узлы, удаляйте узлы и связи
- **Поиск кратчайшего пути** - нахождение пути с минимальной суммой весов
- **Поиск пути с минимальным количеством рёбер** - нахождение пути с наименьшим числом переходов
- **Визуализация** - подсветка найденного пути (зеленый - начальный/конечный узлы, желтый - промежуточные, красный - связи)
- **Сохранение** - экспорт графа в изображение

## 🛠️ Технологии

- **C++17** - язык программирования
- **Qt 5.14.2** - фреймворк для графического интерфейса
- **STL** - стандартная библиотека C++
- **CMake 3.16+** - система сборки

## 🚀 Сборка и запуск

### Windows (Visual Studio 2022)

1. Установите **Qt 5.14.2** (msvc2017_64) и **Visual Studio 2022**
2. Установите **VC++ Redistributable 2017**
3. Склонируйте репозиторий
4. Откройте папку с проектом в Visual Studio (не .sln файл)
5. Настройте путь к Qt в CMake (при необходимости)
6. Соберите проект (Ctrl+Shift+B)

### Linux

```bash
# Установка зависимостей
sudo apt install qt5-default cmake build-essential

# Сборка
mkdir build && cd build
cmake ..
make

# Запуск
./GraphBuilder
```

## 📖 Использование

### Кнопки управления

| Кнопка | Действие |
|--------|----------|
| **Узел** | Создать новый узел |
| **Связь** | Включить режим создания связей |
| **Переместить** | Включить режим перемещения узлов |
| **Удалить узел** | Включить режим удаления узлов |
| **Удалить связь** | Включить режим удаления связей |
| **Кратч. путь** | Поиск пути с минимальной суммой весов |
| **Мин. кол-во ребер** | Поиск пути с минимальным количеством переходов |
| **Очистить всё** | Полная очистка графа |
| **Очистить формат** | Сбросить подсветку |

### Пошаговая инструкция

#### Создание узла

* Нажмите кнопку **"Узел"**
* Новый узел появится в центре сцены

#### Создание связи

* Нажмите кнопку **"Связь"**
* Кликните на первый узел (он станет зелёным)
* Кликните на второй узел
* В появившемся диалоге введите вес связи

#### Перемещение узла

* Просто перетащите узел мышью

#### Удаление узла

* Нажмите кнопку **"Удалить узел"**
* Кликните на удаляемый узел

#### Удаление связи

* Нажмите кнопку **"Удалить связь"**
* Кликните на стрелку (связь)

#### Поиск кратчайшего пути

* Нажмите **"Кратч. путь"** (по весам) или **"Мин. кол-во ребер"** (по количеству переходов)
* Кликните на начальный узел
* Кликните на конечный узел
* Найденный путь подсветится на сцене
* Появится окно с длиной найденного пути

### Управление

| Действие | Результат |
|----------|-----------|
| Левая кнопка мыши | Выбор узлов/связей (в зависимости от режима) |
| Правая кнопка мыши | Отмена текущего режима |

### Визуализация пути

* 🟢 **Зелёные узлы** - начало и конец пути
* 🟡 **Жёлтые узлы** - промежуточные узлы
* 🔴 **Красные связи** - рёбра на пути
* 🔴 **Красные числа** - веса рёбер на пути

## 📝 Примечания
Граф ориентированный (направленные связи)

Веса связей - целые числа от 1 до 1000
