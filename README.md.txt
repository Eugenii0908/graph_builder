Визуальный редактор графов с возможностью создания узлов, связей, поиска кратчайших путей и визуализации результатов.

## 📋 Возможности

- **Создание узлов** - добавляйте узлы на сцену одним кликом
- **Создание связей** - соединяйте узлы направленными связями с указанием веса
- **Редактирование** - перемещайте узлы, удаляйте узлы и связи
- **Поиск кратчайшего пути** - нахождение пути с минимальной суммой весов
- **Поиск пути с минимальным количеством рёбер** - нахождение пути с наименьшим числом переходов
- **Визуализация** - подсветка найденного пути (зеленый - начальный/конечный узлы, желтый - промежуточные, красный - связи)
- **Сохранение** - экспорт графа в изображение

## 🛠️ Технологии

- **C++17** - язык программирования
- **Qt 5.14.2** - фреймворк для графического интерфейса
- **STL** - стандартная библиотека C++
- **CMake 3.16+** - система сборки

## 🚀 Сборка и запуск

### Windows (Visual Studio 2022)

1. Установите **Qt 5.14.2** (msvc2017_64) и **Visual Studio 2022**
2. Установите **VC++ Redistributable 2017**
3. Склонируйте репозиторий
4. Откройте папку с проектом в Visual Studio (не .sln файл)
5. Настройте путь к Qt в CMake (при необходимости)
6. Соберите проект (Ctrl+Shift+B)

### Linux

```bash
# Установка зависимостей
sudo apt install qt5-default cmake build-essential

# Сборка
mkdir build && cd build
cmake ..
make

# Запуск
./GraphBuilder

📖 Использование
Основные элементы интерфейса
Узел - создание нового узла

Связь - режим создания направленных связей

Переместить - режим перемещения узлов (по умолчанию)

Удалить узел - режим удаления узлов

Удалить связь - режим удаления связей

Кратчайший путь - поиск пути с минимальной суммой весов

Минимальное количество ребер - поиск пути с минимальным количеством переходов

Очистить всё - полная очистка графа

Очистить формат - сброс подсветки

Работа с графом
Создание узла: нажмите "Узел" → узел появится в центре

Создание связи:

Нажмите "Связь"

Кликните на первый узел (станет зеленым)

Кликните на второй узел

Введите вес связи

Перемещение: просто перетаскивайте узлы мышью

Удаление узла: нажмите "Удалить узел" → кликните на узел

Удаление связи: нажмите "Удалить связь" → кликните на стрелку

Поиск пути:

Нажмите "Кратчайший путь" или "Минимальное кол-во ребер"

Выберите начальный и конечный узлы

Результат подсветится на сцене

Появится окно с длиной пути

Управление
Левая кнопка мыши - выбор узлов/связей в зависимости от режима

Правая кнопка мыши - отмена текущего режима


📝 Примечания
Граф ориентированный (направленные связи)

Веса связей - целые числа от 1 до 1000
